Tổng Quan Về Cách Một Trang Web Hoạt Động

Để tạo ra một trang web, bạn không cần phải là một thiên tài máy tính. Về bản chất, trang web được cấu thành từ các tệp văn bản chứa các đoạn mã đặc biệt, được trình duyệt web (như Chrome, Edge) đọc và dịch lại thành giao diện hình ảnh trực quan.

Một trang web cơ bản gồm những gì?

Hầu hết mọi trang web tĩnh trên mạng internet đều được xây dựng dựa trên sự kết hợp chặt chẽ của bộ ba ngôn ngữ cốt lõi:

  • HTML: Đóng vai trò như bộ xương, định hình các thành phần (văn bản, nút bấm, ảnh).
  • CSS: Đóng vai trò như lớp áo khoác, sơn màu, căn chỉnh bố cục cho đẹp mắt.
  • JavaScript: Đóng vai trò như hệ thống thần kinh, giúp trang web xử lý hành động phản hồi người dùng.
💡 Hãy nhớ rằng: Bạn hoàn toàn có thể tự làm tất cả những điều này trên một chiếc máy tính cá nhân thông thường mà không tốn bất kỳ chi phí nào!

Bước 1: Chuẩn Bị Môi Trường Làm Việc

Trước khi bắt đầu gõ những dòng mã đầu tiên, bạn cần cài đặt một công cụ viết code chuyên nghiệp thay vì dùng Notepad mặc định của máy tính.

1. Cài đặt VS Code (Visual Studio Code)

Đây là phần mềm soạn thảo mã nguồn phổ biến nhất thế giới hiện nay. Nó hoàn toàn miễn phí, nhẹ và hỗ trợ tự động gợi ý code cực kỳ thông minh.

2. Cách tạo file dự án đầu tiên

  1. Tạo một thư mục mới ngoài màn hình máy tính của bạn và đặt tên là my-website.
  2. Mở phần mềm VS Code lên, kéo và thả thư mục vừa tạo vào giao diện VS Code.
  3. Nhấn vào biểu tượng New File trong VS Code, tạo một file mới đặt tên chính xác là index.html.

Bước 2: Xây Dựng Khung Xương Bằng HTML

HTML sử dụng các "thẻ" (tags) nằm trong cặp dấu < > để thông báo cho trình duyệt biết thành phần đó là gì.

Cấu trúc một file HTML tiêu chuẩn

Hãy sao chép đoạn mã nền tảng này vào file index.html bạn vừa tạo trong VS Code:

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Trang Web Đầu Tiên Của Tôi</title>
</head>
<body>
    <h1>Xin chào! Tôi là một trang web mới.</h1>
    <p>Đây là dòng văn bản giới thiệu đơn giản bằng HTML.</p>
    <button id="my-button">Click Vào Tôi</button>
</body>
</html>
🔍 Cách xem thành quả: Bạn chỉ cần mở thư mục ngoài màn hình máy tính, bấm đúp chuột vào file index.html. Trình duyệt Chrome hoặc Edge sẽ mở ra và hiển thị tiêu đề cùng nút bấm vừa viết!

Bước 3: Trang Trí Giao Diện Bằng CSS

Nếu không có CSS, trang web của bạn trông sẽ rất thô sơ với nền trắng và chữ đen nền tảng. CSS giúp thổi hồn vào giao diện bằng màu sắc và bố cục.

Cách nhúng CSS vào HTML

Để đơn giản nhất cho người mới bắt đầu, chúng ta sử dụng thẻ <style> đặt ngay bên trong cặp thẻ <head> của file HTML:

<style>
    body {
        background-color: #0f172a; /* Đổi màu nền tối */
        color: #ffffff;            /* Đổi chữ sang màu trắng */
        font-family: sans-serif;   /* Đổi font chữ dễ nhìn hơn */
        text-align: center;        /* Căn giữa toàn bộ nội dung */
        padding-top: 50px;
    }
    
    button {
        background-color: #38bdf8; /* Màu nền nút xanh dương */
        color: #1e293b;            /* Màu chữ trong nút */
        padding: 10px 20px;        /* Độ rộng của nút */
        border: none;              /* Bỏ viền thô kệch */
        border-radius: 6px;        /* Bo tròn góc nút */
        cursor: pointer;           /* Biến con trỏ chuột thành hình bàn tay */
        font-weight: bold;
    }
</style>

Bước 4: Tạo Tính Năng Tương Tác Bằng JavaScript

JavaScript là ngôn ngữ lập trình giúp trang web của bạn có thể phản hồi lại các hành động của người dùng (ví dụ: gửi dữ liệu, mở menu ẩn, hiện thông báo alert).

Ví dụ bắt sự kiện click chuột

Đặt thẻ <script> này vào vị trí ngay phía trên thẻ đóng </body> ở cuối file HTML của bạn:

<script>
    // Tìm phần tử nút bấm thông qua ID đã đặt ở phần HTML
    const nutBam = document.getElementById('my-button');

    // Lắng nghe hành động khi người dùng click chuột vào nút
    nutBam.addEventListener('click', function() {
        alert('Chúc mừng! Bạn đã kích hoạt thành công mã JavaScript đầu tiên!');
    });
</script>

Bước 5: Phát Hành Trang Web Lên Mạng Internet

Hiện tại, trang web của bạn mới chỉ chạy offline trên máy tính cá nhân của bạn (Local). Để người khác có thể truy cập được từ xa, bạn cần đưa nó lên một máy chủ lưu trữ (Hosting).

Các bước đưa lên mạng miễn phí bằng GitHub Pages:

  1. Đăng ký tài khoản và tạo một kho lưu trữ mới (Repository) ở trạng thái Public trên trang web GitHub.
  2. Sử dụng phần mềm quản lý mã nguồn Git để đẩy toàn bộ thư mục chứa file index.html lên GitHub bằng các lệnh: 
    git add .
git commit -m "Đăng tải trang web đầu tiên"
git push
  3. Trên giao diện web của GitHub, vào mục Settings của kho lưu trữ > chọn menu Pages bên góc trái > ở phần branch, chọn nhánh main rồi bấm Save.

Hệ thống sẽ mất khoảng 1 phút để xử lý và cung cấp cho bạn một đường link trang web dạng công khai miễn phí. Giờ đây bạn đã có thể gửi link đó cho bạn bè ở bất cứ đâu kiểm tra!
