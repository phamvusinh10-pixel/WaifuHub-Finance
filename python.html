1. Tổng Quan & Cơ Chế Khối Lệnh Của Python

Không giống như đa số các ngôn ngữ khác sử dụng dấu ngoặc nhọn { } để gom cụm mã nguồn, Python bắt buộc người lập trình sử dụng **Khoảng thụt lề (Indentation)**. Đây là đặc điểm lớn nhất giúp mã nguồn Python luôn sạch sẽ và đồng bộ giữa mọi lập trình viên.

Quy tắc Thụt lề sống còn:

Một khối lệnh (Block) trong Python bắt đầu bằng một dấu hai chấm :, và tất cả các dòng lệnh con bên trong khối đó phải được thụt lề bằng nhau (khuyến nghị dùng 4 dấu cách - Spaces hoặc 1 phím Tab).

if True:
    print("Dòng này hợp lệ")  # Thụt vào 4 khoảng trắng
  print("LỖI CÚ PHÁP!")       # Thụt lề sai sẽ gây ra lỗi IndentationError

2. Xử Lý Chuỗi Ký Tự (Strings) Chuyên Sâu

Chuỗi ký tự là một tập hợp các ký tự nằm trong dấu nháy đơn hoặc nháy kép. Python cung cấp một hệ thống các hàm (Phương thức) xử lý chuỗi cực mạnh:

Tên hàm Công dụng xử lý Ví dụ minh họa
upper() / lower() Chuyển đổi toàn bộ chuỗi thành chữ HOA hoặc chữ thường. "abc".upper() $\rightarrow$ "ABC"
strip() Loại bỏ toàn bộ khoảng trắng thừa ở đầu và cuối chuỗi dữ liệu. " hello ".strip() $\rightarrow$ "hello"
replace(old, new) Tìm kiếm cụm ký tự cũ và thay thế hoàn toàn bằng cụm mới. "A B C".replace(" ", "-") $\rightarrow$ "A-B-C"
split(char) Cắt chuỗi thành một Danh sách (List) dựa trên ký tự phân tách. "a,b,c".split(",") $\rightarrow$ ['a', 'b', 'c']
join(list) Nối một danh sách các chuỗi lại thành một chuỗi duy nhất. "-".join(['a', 'b']) $\rightarrow$ "a-b"

Mã thực hành tổng hợp về Chuỗi:

data = "   Họ và Tên: Lâm Xuyên   "
clean_data = data.strip() # Loại bỏ dấu cách thừa
print(clean_data) # Kết quả: "Họ và Tên: Lâm Xuyên"

# Tách chuỗi dựa vào dấu hai chấm
parts = clean_data.split(":")
print(parts) # Kết quả: ['Họ và Tên', ' Lâm Xuyên']

3. Làm Chủ Cấu Trúc Dữ Liệu List (Danh Sách)

List là một cấu trúc dữ liệu dùng để lưu trữ nhiều phần tử trong một biến duy nhất. Các phần tử trong List có thứ tự (Index bắt đầu từ số 0) và có thể thay đổi giá trị được.

Các hàm thao tác với List không thể quên:

fruits = ["Táo", "Chuối", "Cam"]

# 1. len(): Lấy số lượng phần tử đang có trong danh sách
print(len(fruits))  # Kết quả: 3

# 2. append(): Thêm một phần tử mới vào vị trí cuối cùng của List
fruits.append("Xoài")
print(fruits)  # Kết quả: ["Táo", "Chuối", "Cam", "Xoài"]

# 3. insert(index, value): Thêm phần tử vào một vị trí chỉ định trước
fruits.insert(1, "Dâu tây") # Đưa "Dâu tây" vào vị trí số 1
print(fruits)  # Kết quả: ["Táo", "Dâu tây", "Chuối", "Cam", "Xoài"]

# 4. remove(value): Tìm và xóa phần tử đầu tiên khớp với giá trị truyền vào
fruits.remove("Chuối")

# 5. pop(index): Xóa và trả về phần tử tại vị trí chỉ định (mặc định là cuối cùng)
last_item = fruits.pop()
print(last_item)  # Kết quả: "Xoài"

# 6. sort(): Sắp xếp các phần tử trong danh sách theo thứ tự bảng chữ cái hoặc tăng dần
fruits.sort()
print(fruits)  # Kết quả danh sách đã được sắp xếp

4. Cấu Trúc Dictionary (Từ Điển Key - Value)

Dictionary là cấu trúc dữ liệu cực kỳ tối ưu, lưu trữ thông tin dưới dạng cặp Khóa : Giá trị (Key : Value). Mỗi một Key trong Dictionary phải là duy nhất.

Mã nguồn mẫu và các phương thức truy vấn:

user_profile = {
    "name": "Lâm Xuyên",
    "age": 21,
    "role": "Admin"
}

# 1. get(key): Lấy giá trị an toàn, không sợ bị lỗi treo chương trình nếu Key không tồn tại
print(user_profile.get("age"))     # Kết quả: 21
print(user_profile.get("status"))  # Không có key này -> trả về None (thay vì báo lỗi sập hệ thống)

# 2. keys(): Lấy ra danh sách tất cả các Khóa (Keys) đang có
print(user_profile.keys())    # Kết quả: dict_keys(['name', 'age', 'role'])

# 3. values(): Lấy ra danh sách tất cả các Giá trị (Values) tương ứng
print(user_profile.values())  # Kết quả: dict_values(['Lâm Xuyên', 21, 'Admin'])

# 4. items(): Trả về cặp bộ (Key, Value) dùng rất nhiều trong vòng lặp
for key, value in user_profile.items():
    print(key, "=>", value)

5. Điều Khiển Luồng Tối Ưu Với Vòng Lặp

Khi kết hợp các hàm toán học và cấu trúc tuần hoàn, bạn có thể giải quyết các bài toán dữ liệu lớn chỉ bằng vài dòng mã ngắn.

Hàm range() và kỹ thuật duyệt List nâng cao

# Hàm range(start, stop, step) sinh ra chuỗi số liên tục
for i in range(1, 10, 2):
    print(i) # In ra các số lẻ: 1, 3, 5, 7, 9

# Kỹ thuật duyệt kết hợp hàm enumerate() để lấy cả vị trí (index) lẫn giá trị
names = ["An", "Bình", "Cường"]
for index, name in enumerate(names):
    print(f"Vị trí {index} là bạn: {name}")
💡 Từ khóa Break & Continue: Dùng break để lập tức thoát hoàn toàn khỏi vòng lặp trước thời hạn. Dùng continue để bỏ qua lượt chạy hiện tại và nhảy sang lượt lặp tiếp theo ngay lập tức.

6. Xử Lý Tập Tin Văn Bản (File I/O Handling)

Để lưu trữ dữ liệu lâu dài không bị mất khi tắt máy, chương trình Python cần đọc hoặc ghi thông tin xuống các file trên ổ cứng (như tệp .txt, .csv).

Kỹ thuật sử dụng khối lệnh `with open` an toàn

Trong Python, thay vì dùng cách mở file truyền thống rồi phải nhớ đóng lại, lập trình viên chuyên nghiệp luôn dùng từ khóa with. Khi khối lệnh kết thúc, Python sẽ tự động đóng file giải phóng bộ nhớ cho hệ điều hành kể cả khi xảy ra lỗi đột ngột.

# 1. Ghi dữ liệu vào file (Chế độ 'w' - Write, sẽ ghi đè toàn bộ file)
with open("report.txt", "w", encoding="utf-8") as file:
    file.write("Dòng dữ liệu thứ nhất.\n")
    file.write("Dòng dữ liệu thứ hai.\n")

# 2. Đọc dữ liệu từ file (Chế độ 'r' - Read)
with open("report.txt", "r", encoding="utf-8") as file:
    content = file.read() # Đọc toàn bộ nội dung file đưa vào một biến chuỗi
    print(content)

# 3. Đọc file theo từng dòng đưa vào một List bằng hàm readlines()
with open("report.txt", "r", encoding="utf-8") as file:
    lines = file.readlines()
    for line in lines:
        print("Đọc từng dòng:", line.strip())
