Bài 1: Giới thiệu & Cài đặt Python

Python là một ngôn ngữ lập trình bậc cao, cực kỳ phổ biến nhờ vào cú pháp rõ ràng, dễ đọc và gần gũi với tiếng Anh bản xứ. Đây là lựa chọn hoàn hảo hàng đầu cho người mới bắt đầu học lập trình.

Tại sao nên học Python?

Python được ứng dụng rộng rãi trong nhiều lĩnh vực: Tự động hóa văn phòng (Excel, Word), Phân tích dữ liệu, Trí tuệ nhân tạo (AI), và Phát triển Web.

Chương trình đầu tiên: Hello World

Để xuất một dòng chữ ra màn hình trong Python, bạn chỉ cần sử dụng hàm print():

# Đây là dòng ghi chú (comment), Python sẽ bỏ qua nó
print("Xin chào thế giới! Tôi đang học Python.")
Thực hành: Bạn có thể truy cập trang web repl.it hoặc online-python.com để chạy thử đoạn code trên trực tiếp ngay trên trình duyệt mà chưa cần cài đặt phần mềm phức tạp vào máy.

Bài 2: Biến và Kiểu dữ liệu

Biến (Variable) giống như một chiếc hộp dùng để lưu trữ dữ liệu. Trong Python, bạn không cần phải khai báo kiểu dữ liệu trước, Python sẽ tự động nhận diện giúp bạn.

Khai báo biến

ten = "Lâm Xuyên"  # Kiểu chuỗi (String)
tuoi = 21          # Kiểu số nguyên (Integer)
diem = 8.5         # Kiểu số thực (Float)
is_learning = True # Kiểu logic (Boolean)

Cách kiểm tra kiểu dữ liệu

Bạn có thể dùng hàm type() để kiểm tra xem biến đó thuộc kiểu dữ liệu nào:

print(type(tuoi))  # Kết quả sẽ trả về: <class 'int'>

Bài 3: Các Toán tử cơ bản

Python hỗ trợ đầy đủ các phép toán số học cơ bản tương tự như các máy tính bỏ túi thông thường.

Toán tử số học

a = 10
b = 3

print(a + b)  # Phép cộng -> 13
print(a - b)  # Phép trừ -> 7
print(a * b)  # Phép nhân -> 30
print(a / b)  # Phép chia lấy cả phần thập phân -> 3.333333
print(a // b) # Phép chia lấy phần nguyên -> 3
print(a % b)  # Phép chia lấy phần dư -> 1
print(a ** b) # Phép lũy thừa (10 mũ 3) -> 1000

Bài 4: Câu lệnh điều kiện (If - Else)

Câu lệnh điều kiện giúp chương trình của bạn có khả năng "tư duy" và đưa ra các quyết định xử lý khác nhau dựa trên từng trường hợp cụ thể.

Quy tắc quan trọng: Python sử dụng khoảng thụt lề (Indentation - thường là 4 dấu cách hoặc 1 phím Tab) để xác định khối lệnh. Nếu thụt lề sai, chương trình sẽ báo lỗi ngay lập tức.

Ví dụ cấu trúc If - Elif - Else

diem = 7.5

if diem >= 8.0:
    print("Học lực: Giỏi")
elif diem >= 6.5:
    print("Học lực: Khá")
else:
    print("Học lực: Trung bình/Yếu")

Bài 5: Vòng lặp (Loops)

Vòng lặp được sử dụng khi bạn muốn lặp đi lặp lại một hành động hoặc một khối lệnh nào đó nhiều lần để tối ưu hóa công sức viết code.

1. Vòng lặp For

Thường dùng khi bạn đã biết rõ số lần lặp cụ thể hoặc muốn duyệt qua một danh sách dữ liệu.

# Chạy vòng lặp từ 0 đến 4 (5 lần)
for i in range(5):
    print("Lần lặp thứ:", i)

2. Vòng lặp While

Chạy liên tục chừng nào điều kiện kiểm tra bên dưới vẫn còn đúng (True).

dem = 1
while dem <= 3:
    print("Số hiện tại là:", dem)
    dem = dem + 1 # Tăng biến đếm để tránh vòng lặp vô hạn

Bài 6: Hàm trong Python (Functions)

Hàm là một khối lệnh được gom nhóm lại với nhau để thực hiện một tác vụ cụ thể. Hàm giúp bạn tái sử dụng code nhiều lần mà không cần phải viết lại từ đầu.

Cách định nghĩa và gọi hàm

Sử dụng từ khóa def để bắt đầu khai báo một hàm mới:

# Định nghĩa hàm tính tổng có nhận vào 2 tham số
def tinh_tong(a, b):
    ket_qua = a + b
    return ket_qua # Trả về giá trị sau khi tính xong

# Gọi hàm và sử dụng kết quả
ket_qua_1 = tinh_tong(5, 7)
print("Tổng hai số là:", ket_qua_1) # Kết quả xuất ra: 12
